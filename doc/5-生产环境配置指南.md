# 🔧 生产环境配置指南

## 配置优先级

```
环境变量 > .env文件 > config.json > 默认值
```

---

## 方式一：管理后台配置（推荐）

### 访问管理后台

1. 打开浏览器访问：`http://你的IP:5000/admin`
2. 输入管理密码（默认：`admin123`）
3. 在页面中配置AI参数
4. 点击"保存配置"

### 优点
- ✅ 无需重启服务，即时生效
- ✅ 可视化界面，操作简单
- ✅ 支持测试连接功能
- ✅ 自动保存到config.json

---

## 方式二：配置文件（config.json）

### 创建配置文件

在项目根目录创建 `config.json`：

```json
{
  "AI_API_URL": "https://api.deepseek.com/v1/chat/completions",
  "AI_API_KEY": "sk-xxx…xxxx",
  "AI_MODEL": "deepseek-chat",
  "ADMIN_PASSWORD": "your-secure-password"
}
```

## 方式三：环境变量

### Linux/Mac

```bash
# 临时设置（当前终端有效）
export AI_API_URL=https://api.deepseek.com/v1/chat/completions
export AI_API_KEY=sk-xxx…xxxx
export AI_MODEL=deepseek-chat
export ADMIN_PASSWORD=your-secure-password

# 启动服务
python3 app.py
```

### 永久设置（添加到 ~/.bashrc）

```bash
echo 'export AI_API_URL=https://api.deepseek.com/v1/chat/completions' >> ~/.bashrc
echo 'export AI_API_KEY=sk-xxx…xxxx' >> ~/.bashrc
echo 'export AI_MODEL=deepseek-chat' >> ~/.bashrc
source ~/.bashrc
```

### Docker环境

```bash
docker run -e AI_API_URL=https://api.deepseek.com/v1/chat/completions \
           -e AI_API_KEY=sk-xxx…xxxx \
           -e AI_MODEL=deepseek-chat \
           -p 5000:5000 \
           math-site
```

## 方式四：.env文件

### 创建.env文件

在项目根目录创建 `.env` 文件：

```env
# AI配置
AI_API_URL=https://api.deepseek.com/v1/chat/completions
AI_API_KEY=sk-xxx…xxxx
AI_MODEL=deepseek-chat

# 管理密码
ADMIN_PASSWORD=your-secure-password
```

### 注意事项

- `.env` 文件不会被Git提交（已在.gitignore中）
- 修改后需要重启服务
- 不要在.env中使用引号

## 生产环境部署建议

### 1. 修改管理密码

```json
{
  "ADMIN_PASSWORD": "使用强密码，包含大小写字母、数字、特殊字符"
}
```

### 2. 使用环境变量（最安全）

```bash
# 在服务器上设置环境变量
export AI_API_KEY=sk-xxx…xxxx
export ADMIN_PASSWORD=***
```

### 3. 使用密钥管理服务

对于大规模部署，建议使用：
- AWS Secrets Manager
- HashiCorp Vault
- 阿里云密钥管理服务

### 4. 限制API Key权限

在AI服务后台：
- 设置使用限额
- 限制IP访问
- 定期轮换Key

## 配置项说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `AI_API_URL` | AI服务API地址 | DeepSeek地址 |
| `AI_API_KEY` | API密钥 | 空（离线模式） |
| `AI_MODEL` | 模型名称 | deepseek-chat |
| `ADMIN_PASSWORD` | 管理后台密码 | admin123 |
| `SECRET_KEY` | Flask密钥 | 随机生成 |
| `DATABASE_URI` | 数据库地址 | SQLite文件 |

## 常见问题

### Q1: 配置后不生效？

**A:** 检查优先级，环境变量会覆盖配置文件。

### Q2: 如何确认配置已加载？

**A:** 启动时会显示：
```
🤖 AI模式: 已配置
```

### Q3: 如何切换回离线模式？

**A:** 清空 `AI_API_KEY` 即可。

### Q4: 忘记管理密码怎么办？

**A:** 删除 `config.json` 中的 `ADMIN_PASSWORD` 字段，使用默认密码。
