## 一、项目概述

### 1.1 项目背景

- 湖北省每年约有 **60-70万** 幼儿园大班儿童面临幼升小
- 武汉市作为省会城市，每年幼升小适龄儿童约 **12-15万**
- 家长对幼小衔接培训需求旺盛，但受政策限制（严禁幼儿园"小学化"），合规的思维训练类产品存在巨大市场空白
- 现有产品（豌豆思维、火花思维等）价格较高，且依赖网络，本地化服务不足

### 1.2 产品定位

| 项目 | 原定位 | 新定位 |
|------|--------|--------|
| 产品名称 | 数学练习网站 | **数苗乐园 — 幼小衔接思维训练平台** |
| 核心价值 | 题库练习 | **AI驱动的个性化思维训练** |
| 目标用户 | 通用 | **5-7岁幼小衔接阶段儿童及家长** |
| 合规表述 | 数学练习 | **思维能力训练、学习习惯培养** |

### 1.3 竞争优势

1. ✅ **合规安全** — 定位"思维训练"而非"学科培训"，符合教育部去小学化政策
2. ✅ **离线可用** — 无需持续网络连接，适合家庭场景
3. ✅ **价格亲民** — 远低于线上竞品（豌豆思维4000-8000元/年）
4. ✅ **本地化** — 更了解湖北教育特点和家长需求
5. ✅ **AI赋能** — 接入大模型，实现个性化出题和智能辅导

```python
    """AI智能出题引擎"""
    
    def generate_questions(self, user_id, difficulty='auto'):
        # 1. 分析用户历史数据
        history = self.analyze_user_history(user_id)
        
        # 2. 确定难度等级
        if difficulty == 'auto':
            difficulty = self.calculate_difficulty(history)
        
        # 3. 薄弱知识点加强
        weak_points = self.identify_weak_points(history)
        
        # 4. 生成题目（本地规则 + AI增强）
        questions = []
        questions.extend(self.generate_calc(difficulty, weak_points))
        questions.extend(self.generate_pattern(difficulty))
        questions.extend(self.generate_compare(difficulty))
        questions.extend(self.generate_count(difficulty))
        questions.extend(self.generate_word_problem(difficulty, weak_points))
        
        return questions
    
    def analyze_user_history(self, user_id):
        """分析用户答题历史"""
        recent_answers = Answer.query.filter_by(user_id=user_id)\
            .order_by(Answer.answered_at.desc()).limit(100).all()
        
        return {
            'avg_score': calculate_average(recent_answers),
            'accuracy_by_type': group_accuracy_by_type(recent_answers),
            'speed_stats': calculate_speed_stats(recent_answers),
            'weak_types': identify_weak_types(recent_answers)
        }
```
