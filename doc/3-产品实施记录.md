## 🎮 任务1：游戏化系统

### 需求
为网站添加积分系统、徽章系统和排行榜，提升用户粘性。

### 实现内容

#### 数据库新增模型
```python
class Achievement(db.Model):
    """成就定义"""
    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(50), nullable=False)  # 成就名称
    icon = db.Column(db.String(10), nullable=False)   # 图标emoji
    description = db.Column(db.String(200))            # 成就描述
    condition_type = db.Column(db.String(30))          # 条件类型
    condition_value = db.Column(db.Integer)            # 条件值
    points_reward = db.Column(db.Integer, default=0)   # 奖励积分

class UserAchievement(db.Model):
    """用户获得的成就"""
    id = db.Column(db.Integer, primary_key=True)
    user_id = db.Column(db.Integer, db.ForeignKey('user.id'))
    achievement_id = db.Column(db.Integer, db.ForeignKey('achievement.id'))
    earned_at = db.Column(db.DateTime, default=datetime.now)

class PointLog(db.Model):
    """积分记录"""
    id = db.Column(db.Integer, primary_key=True)
    user_id = db.Column(db.Integer, db.ForeignKey('user.id'))
    points = db.Column(db.Integer, nullable=False)
    reason = db.Column(db.String(100))
    created_at = db.Column(db.DateTime, default=datetime.now)
```

## 🤖 任务2：AI智能出题

### 需求
保留现有离线可用，增加AI智能出题方式。

### 设计思路
- ✅ 保留现有离线出题（默认）
- ✅ 新增AI智能出题（可选）
- ✅ 根据用户薄弱知识点个性化出题
- ✅ AI不可用时自动降级到离线模式

### 实现内容

#### 出题模式
| 模式 | 说明 | 网络要求 | 成本 |
|------|------|----------|------|
| 📱 离线模式 | 本地随机算法 | ❌ 不需要 | 免费 |
| 🤖 AI智能 | 调用大模型API | ✅ 需要 | 约0.001元/次 |

#### AI出题特点
1. **个性化** — 根据历史答题数据针对薄弱知识点出题
2. **多样化** — 每次生成不同题目
3. **自适应** — 薄弱知识点题目比例增加
4. **自动降级** — AI调用失败时切换到离线模式

#### 支持的AI服务
| 服务 | API地址 | 价格 | 推荐度 |
|------|---------|------|--------|

| OpenAI | api.openai.com | $0.5/百万tokens | ⭐⭐⭐ |

1. **管理后台**（推荐）— http://localhost:5000/admin

#### 界面更新

## 💬 留言板功能

### 需求
让使用的家长可以提出遇到的问题和想法，留言内容过滤掉不文明用语。

### 功能特性

#### 留言类型
- 💬 问题反馈
- 💡 建议想法
- ❓ 疑问咨询

#### 敏感词过滤
**过滤内容：**
- 不文明用语（脏话、粗口）
- 攻击性词汇
- 色情相关词汇
- 广告spam
- 拼音/谐音变体（如 sb、nmsl 等）

**过滤效果：**
- 包含敏感词 → 提示"留言包含不当内容，请修改后重试"
- 不会泄露具体哪个词被过滤

#### 其他特性
- 🙈 匿名发布
- 🏷️ 分类筛选
- 🗑️ 删除留言（只能删除自己的）
- ⭐ 积分奖励（发布留言+5积分）

### 新增页面
- `/messages` — 留言板页面

### 新增API
- `GET /api/messages` — 获取留言列表（支持分页、筛选）
- `POST /api/messages` — 发表留言
- `DELETE /api/messages/<id>` — 删除留言
