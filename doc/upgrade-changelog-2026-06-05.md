# 数苗乐园升级清单（2026-06-05）

## 1. 升级概述

本次升级在原有数学练习、积分、成就、排行榜、错题本、家长看板、留言系统基础上，新增了以下能力：

- 贴纸收集与兑换
- 童话屋房间物品与解释卡
- 轻养成数值（聪明值 / 勇气值 / 耐心值）
- 房间阶段性升级表现
- 家长侧解释说明

核心目标是把“积分”从抽象数字，升级为：
- 可收集
- 可展示
- 可养成
- 可解释

的学习奖励体系。

## 4. 新增系统模块

### 4.1 StickerSystem
功能包括：
- 初始化默认贴纸
- 贴纸图鉴
- 用户贴纸查询
- 积分兑换贴纸

### 4.2 FairyHouseSystem
功能包括：
- 初始化默认房间物品
- 查询童话屋房间状态
- 查询房间物品解释卡
- 根据贴纸数量自动解锁房间物品

### 4.3 GrowthSystem
功能包括：
- 初始化用户养成数值
- 贴纸兑换后增长养成数值
- 按贴纸系列映射成长属性：
  - 计算 / 比较 / 规律 → 聪明值
  - 应用题 → 勇气值
  - 坚持 → 耐心值
- 计算房间阶段：
  - `起点小屋`
  - `成长小屋`
  - `闪光童话屋`

## 9. 本次升级的关键设计思路

### 9.1 不破坏主链路
本次升级没有改变：
- 练题流程
- 积分获得方式
- 排行榜与成就核心逻辑

### 9.2 积分只用于消费，不影响难度
贴纸只做：
- 收集
- 展示
- 养成
- 解释

不改变练习题目难度，保持学习公平性。

### 9.3 养成可视化
通过贴纸与房间阶段，把学习进步拆成三层：
- 短期反馈：贴纸
- 中期反馈：养成数值
- 长期反馈：房间阶段
